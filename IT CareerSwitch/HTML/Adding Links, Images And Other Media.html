    The World Wide Web (www) is all about links to other locations making it a giant playground where you can surf at high speed to different parts of the world

    Hyperlinks give your web page the ability to interact with other sources around it in fantastic ways

    Virtually every web page beyond the simplest example will have at least some form of hyperlink

    In order to create hyperlinks you must have three pieces of information

    Web address

    Some text to display

    An anchor element (a)

    An inline element that should always be placed within a block element

    Used to anchor a URL to some text on your webpage

    Contains an href attribute that ties everything together

    (a href=http://www.google.co.uk)Google!(/a)

    Relative Links

    Use a type of shorted to specify the URL you are pointing to

    Only used in certain situations like linking within the same page or the same site

    When the browser sees a link like (a href="about.html)About(/a) it assumes that because the domain name is missing it is a relative link

    The browser then uses the domain name to locate the resource

    Additional steps are required if the page is in a different directory

    Relative links have the advantage of functioning even if you happen to move the web site to another server or domain!

    URLs can be fairly picky sometimes even being case sensitive so if your URL and (a) element aren't working try these steps

    Check the capitalization

    Check the extension

    Check the filename

    Check the URL syntax http:// is required for absolute links

    Working With Images

    Images play an important role on the web page and can include:

  • Logos
  • Banners
  • Clickable navigation aids
  • Content

    • When well used they are a key element of page design and when used poorly they can result in an underable, unintelligible page and a poor user experience

    The first step to working with images is to determine the appropirate file format and size for images

    Web-friendly images aren't the same as images for other mediums

    Require a smaller size for fast page loading

    Require a cross-platform file type

    Best general use file formats

    GIF - Graphics Interchange Format

    Supports only up to 256colors

    Best for less-complex, non-photographic images like icons, line art or clip art

    JPEG - Joint Photographic Experts Group

    Supperts 24-bit color

    Cross platform and application independent

    Compromises quality when file is compressed though a balance can be struck

    PNG - Portable Network Graphics

    Supports up to 32-bit-color

    Combines the best aspects of GIF and JPEG

    Offers the best balance between quality and file size

    Adding images to the page requires that you know the location of the image and this works best if you can use relative locations

    You could use ablosute source references to images stored elswehere

    Using relative source references is preferred
