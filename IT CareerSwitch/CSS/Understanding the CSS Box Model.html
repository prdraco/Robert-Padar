What is the Box Model

All elements in CSS are treated as a box essentially wraping around every single HTML element on the page

The box consists of four properties:

-Margin

-Border

-Padding

-Content

Height and Width

The height and width properties of the box deal with the dimensions of the content area and are given as # of pixels or percentage

-Height:100px;

-Height:80%

-Height:auto

If the amount of content exceeds the size of the box surrounding its element the content will flow out of the box and result in a badly formatted web page

Use the overflow property to prevent this:

-Visible

-Hidden

-Scroll

-Auto

Margin and Padding

The padding controls the free space around the content inside each box while the margin controls the free space around the outside of the box

Both of these controls are transparent and therefore changing their color is not possible however, the width can be modified to adjust the positioning of elements

Several syntax options exist for both margins and padding

Margin and Padding syntax:

-Margin: 25px; - sets all four margins to 25px

-Margin: 25px 50px - sets the top and bottom margins to 25px and the left and right margins to 25px

-{Margin-top: 25px; margin-right: 50px, margin-bottom: 50px; margin-left: 10px;}

-Margin: 25px 50px 50px 10px

Additionally margins (not padding) can have negative values

Working with Borders

The CSS border properties allow you to set various properties for the border of each element in the CSS box model

-Border-width

-Border-color

-Border-style

-Border-radius

The border-with is the thickness of the border that surrounds each HTML element

-Normally set in pixels

-Provides the alternative of using one of three predefined values:

-Thin

-Medium

-Thick

-Use the same syntax as padding and margins:

-Border-width: 25px - set all borders to 25px

-Border-width: 25px thin - sets the top and bottom to 25px and the left/right to thin

-Border-top-width: 30px

Border-color us used to configure the color of the border

Use one of 140 predefined color names

Use RGB notation

-All web colors are represented by the three primary colors of Red, Green and Blue (RGB)

-RGB (0,255,0) or RGB (255,0,255)

Use hexadecimal notation - uses 6 digits to represent color

- 00 00 00

- FF 00 00

- FF 00 FF

Set to transparent

Border-style is to configure the various border-styles which are availabe in CSS

-none

-Dotted

-Dashed

-Solid

-Double

-Groove

-Ridge

-Inset

-Outset

Border-radius is used to create borders with rounded corners

Values are commonly given in pixel or percentage

Can be set individually for the four corners:

-Top-left

-Top-right

-Bottom-right

-Bottom-left

Examples:

-Border-radius: 5px;
