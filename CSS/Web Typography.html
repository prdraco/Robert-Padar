Understanding Fonts

A font is a set of characters that share a similar design:

-Times New Roman

-Arial

-Verdana

-Comic Sans

Font properties in CSS are used to format the appearance of words and text on a page

The font property is concerned with how the characters look while the text property is concerned with everything else

Font families define the three major categories of font types with two additional types:

-Serif

-San serif

-Monospace

-Cursive

-Fantasy

When specifying the font-family you should always specify multiple fonts in case the user does not have the font you specify

The font-size property sets the size of the text and can be set in multiple ways:

-Pixels (px)

-Em

-Percentage (%)

-Keywords

Examples:

-Font-size: 20px;

-Font-size: 1.5em

-Font-size: 150%;

-Font-size: large

Font-style property - used to specify different styles:

-Normal

-Italic

-Oblique

Font-weight property - used to specify bold text and works as the counterpart to font-style:

-Normal, bold, or bolder (some browsers may not support bolder)

-Numbers 100-900 (above 600 is considered bold

Working With Text Properties

CSS text properties allow you to set properties that aren't realated to the front of the text:

-Text color

-Text alignment

-Text decoration

-Letter spacing

-Word spacing

-Line height

The text-alignment property is used to specify how text should be aligned within an element on the page:

-Left

-Right

-Centered

-Justified:

-Text-align:center

-Text-align:left

-Left alignment is the default

The text-decoration property is mainly used to specify whether or not text should be underlined though multiple values are possible for the property:

-None - default

-Underline

-Overline

-Line-through

This is most commonly used to remove underlines from hyperlinks which will occur by default in most browsres

Text-decoration:none;

Letter spacing is used to increase or decrease the spacing between letters in a word

The amount of spacing is specified in pixels:

-To increase space use a positive value

-To decrease space use a negative value:

-Letter-spacing: 2px;

Word spacing is used to increase or decrease the spacing between the actual words in a block of text:

-Similar in function to letter-spacing

-Positive values increase space and negative values decrease space:

-Word-spacing: 2px;

Line height is used to set the spacing between each line of text

Can be set using multiple values:

-Number

-Specific length

-Percentage:

-Line-height: 20px;

-Line-height: 120%

Text and Shadow Effects

CSS3 contains many new properties to style, alter, customize, tweak, and enhance text on the page

As a result of these new properies the use of images to achieve desired text effects is greatly reduced and this has a positive effect on page load time and other advantages:

-Text takes up less storage space and downloads faster

-Screen readers and other types of alternative browsers can read text but not images

-Search engines and read and index text

Shadows are a great tool for adding an illusion of depht to a web page

CSS3 provides two properties that can be used to create shadows:

-Text-shadow

-Box-shadow

By applying shadows in different ways you can create many types of effects:

-Drop shadows

-Letterpress text

-3D text

Text-shadow - property used to create text shadows

At its simplest:

-Text-shadow: offset-x offset-y color

-H1 {text-shadow: 0.1em 0.1em #aaaaaa;}

An additional optional parameter is the blur-radius which is used to make the shadow fuzzy and lighter

The larger the value the larger and lighter the blur will be

The box-shadow is used to create one or more shadows for an element and has a similar syntax:

-Box-shadow offset-x offset-y color

-Box-shadow: 4px 4px #777777;

Inset text is text that appears to recede into the background

The key is to use a text shadow that's lighter than the text background color causing the text to look like it's inset and a shadow is being cast inside the letters

The lighter shadow and darker background makes the text look inset

Creating 3D text is the opposite of inset text in which the text appears to protrude from the page

The key is to use multiple tex shadows together along with another CSS3 trick which is opacity

It can get comlicated but the effect is fantastic!

Just to show this special text-shadow effect

You can also use transparency along with a new property called background-clip to do a much more realistic inset otherwise known as a letterpress effect

The most common use of the background-clip property is to add a background image to text which creates some stunning effects

This property specifies whether the background extends under the border of the object
