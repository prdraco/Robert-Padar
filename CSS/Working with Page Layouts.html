Positioning Elements

As the name implies the CSS positioning property is used to configure the location of the element on the page

In the case of an overlap the positioning element specifies which element should be on top

Four separate methods exist for positioning elements

Static positioning:

-Elements are positioned according to the normal flow of the page

-All HTML elements use static positioning by default

-There generally isn't a reason to specify this type of positioning unless you are attempting to override another style rule

Relative positioning is a method that positions an element "relative to its normal position":

-Normal is the default position when no positioning rule is specified

-Normal is also used when static positioning is in place

So if an element would normally be directly beneath the precending element using relative positioning will change that position in comparison with the normal position

Relative position can result in elements that overlap in which case you have to use the z-index property

Element with the greater z-index value be posotioned in front

Fixed positioning means that an element will always stay at its assigned location and will not move even when the page is scrolled:

-Headers

-Social sharing buttons

-Footers

When using fixed positioning you can use the top property to specify the number of pixels the element should be from the top of the page and the left property for distance from left of the page

Floating Elements

Floating is a technique used to arrange elements on a page by taking them out of the normal flow:

-Float: Left; - pushes the element to the left the furthest it can go

-Float: Right; - pushes the element to the right the furthest it can go

Items that are not configured to float other object will follow normal flow and flow around the floated element

Controlling Display and Visibility

In addition to controlling the positioning of elements using position and float you can control the visibility of those elements on the page as well

The properties allow you to remoce or hide elements when necessary:

-Display

-Visibility

The two CSS properties that contorl wheter or not elements apper and how they apper are DISPLAY and VISIBILITY

Display contols how the element is displayed and has three possible values:

-None - the element dissapears from the page

-Inline - the element is displayed as an inline element and thus does not start and end with a new line.

-Block - the element is displayed as a block element and starts and ends with a new line

Visibility can be used to hide an element:

-Visibility: Hidden;

-This hides the elements but it is still present on the page
