Using Transforms

The translate value of the transform property will reposition the object

Translate will occur to the right by default

Translate occurs on the X axis by default:

- -webkit-transform:translate(90px);

- -To be explicit you can use translateX or translateY or specify both

webkit-transform:translate(100px, 50px);

The transform property in CSS3 allows you advanced management of web page element:

-Rotation

-Scale

-Skew

-Reposition

All of this while the margins are left intact results in the rest of the page being left intact

Transforms may not be supported by every browser that acces your site so you should check for support for implementing on your site

Two types of transforms exist

-2D

-3D

Transitions

Transitions describe functional and gradual state changes to objects on the web page and are possible with CSS3

Transitions are essentially a rudimentary from of animation

Any element on the page can have transitions applied but common implementation is a hyperlink that appears as a tab which transitions into the page in order to be selected

To create a transition effect, you must specify two thing:

-the CSS property you want to add an effect to

-the duration of the effect

This allows you to change properties values smoothly over a period of time:

div {

width: 100px;

height: 100px;

background: red;

-webkit-transition: width 2s; /*safari*/

transition: width 2s

}

Using Animations

CSS allows designers to apply animations to HTML objects with describe how the element changes from one state to another

Animations also provide the abiltiy to determine what the states of animation will be

Animations are considered as a series of keyframes each of which is a state the object will pass through over the course of its animation

Animations are configured using at @ rule with the keyword keyframes:

-@-webkit-keyframes colorChange {from {background-color:red;} to {background-color:blue;}}

-then the animation is configured for a particular element by calling the name of the function using -webkit-animation:colorChange and specifying a timeframe
