    DOCTYPE Declaration

    Defines the version of HTML that you will be using

    (!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd")

    (!DOCTYPE html)

    Allows for the use of markup validators

    Standard portion of all HTML documents

    The body

    The body of the HTML document is the portion that defines all the important content of your web page

    What you want your visitors to see is all found in the body

    Includes structure to define the content

    Various elements and attributes:

  • Headings
  • Paragraphs
  • Sections
  • Lists
  • Images and media

    • Can include styling of the content
